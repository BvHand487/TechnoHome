SmartHome API

An API that provides endpoints for retrieving and modifying sensor data.

API Documentation

You can find detailed documentation for our API in the Swagger UI:

  • API Swagger Documentation

Endpoints

  • GET /api/records: Retrieve all records.
  • GET /api/records?type={type: string}&id={id: int}: Retrieve records using filters.
  • DELETE /api/records?date={date: datetime}: Deletes records before a given date.

  • GET /api/sensors: Retrieve all sensors.
  • GET /api/sensors/{id: int}: Retrieve a sensor by id.
  • PATCH /api/sensors/{id: int}?enabled={false/0/true/1}&update={update: int}: Configures a sensor by id.
  • DELETE /api/sensors/{id: int}: Deletes a sensor by id.
